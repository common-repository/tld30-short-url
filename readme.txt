Easiest way to create Short URL's for your posts and page permalinks. Tiny URL's for your WordPress blog.

== Description ==  

This a very simple plugin for WordPress. What is does is generate a unique short url for each of your blog posts and pages. This unique short url will be displayed where ever you enter a PhpCode. <a href="http://tld30.com/webmaster/tools/wordpress-short-url-plugin/" target="_blank" title="Install Help">Install Help</a>
